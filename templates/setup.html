PFA uses Claude AI to automatically categorise transactions. To enable it, set your Anthropic API key as a Windows environment variable — it's never stored in a file.

Copy and run this in PowerShell (replace sk-ant-... with your real key):
[Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "sk-ant-...", "User")
After setting it, restart PFA for the new variable to take effect. Don't have a key? Sign up at console.anthropic.com — free tier available. Or skip this; PFA still works using rule-based categorisation.
PFA imports transactions from files you download from your bank's website. Upload them via the ⬆ Import button once you're set up.
Supported banks & file types

PayPal CSV PayPal → Activity → Statements → Download → CSV (all transactions)
Revolut CSV Revolut app → Account → Statement → Export to CSV
