Template Types
TypeDescriptionExamples
Generated (static HTML) Written to disk by reporter.py during import or Refresh Charts. Served as static files. No live data — must re-run reporter to update. Nav links no longer point here for converted pages. monthly_summary.html, fy_summary.html, review.html, transfers.html, net_worth.html, transactions.html (all still generated as backup/CLI output; nav links point to live routes)
Live (server-rendered) Served by server.py Flask routes via Jinja2 templates. Data is read from the DB and JSON files on each request. Always current. Uses {% include 'partials/_nav.html' %} so help drawer, badges, and upload modal propagate automatically. transactions, dashboard, fy-summary, net-worth, review, transfers, subscriptions, commitments, merchant-rules, reimbursements, coverage, cash-flow, financial-goals, loans, recommendations, capital-gains, franking-credits, tags, help
Partials Jinja2 includes shared across all pages. Rendered server-side before serving. templates/partials/_nav.html — nav bar, tab bar, import modal, upload modal, badge JS
